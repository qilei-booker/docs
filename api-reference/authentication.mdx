## Getting Access

You can check all your API quotas and request rate limitation on the "Account" page. There are two types of APIs in ThreatBook CTI, you can use them according to your actual needs.

- **Community API** is public to all our ThreatBook CTI members, there is no need to apply for access for it except if you would like to get extra quotas. please email us at [contactus@threatbook.io](mailto:contactus@threatbook.io) if you have any questions or concerns, it is our pleasure to be able to serve you. 

- **Premium API** is private for commercial customers or particular group. If you would like to get a free trial for them, please email us or join the [Slack channel](/slack/link) to talk with us.

## API key

The unique identifier for API request. You are able to get community API key and premium API key on the "My API" page of [threatbook.io](https://threatbook.io).

## Access IP

Please bind your access IP to the key and check whether you have the authority quotas to access the specific API before you interact with it.

###### Please pay attention:

You are not allowed to abuse our threat intelligence in illegal activities, or unauthorized commercial use. We will disable your API key or account permanently once we find out that you don’t obey our terms of service, and reserve the right to hold you accountable.
